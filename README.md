# ⚡ Vertex Sentinel

> **Verifiable risk-management for autonomous AI trading agents.**
> EIP-712 signed TradeIntents · On-chain guardrails · Fail-Closed execution · No private key delegation.

## 📊 Live Monitoring Dashboard

### How to run the Dashboard:

1. **Start the Monitor**:
   ```bash
   npm run dashboard
   ```

2. **Access the UI**:
   Open http://localhost:3000/dashboard/index.html in your browser.

## 🏗️ Hardened & Verified Architecture

- **Fail-Closed Execution**: Critical failures or risk breaches trigger an immediate halt.
- **Verifiable Audit Trail**: Every trade decision generates a cryptographically signed EIP-712 checkpoint.
